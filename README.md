# ShopEasy E-Commerce App

![Python CI/CD](https://github.com/Mohid107/Ecommerce_tkinter/actions/workflows/python-test.yml/badge.svg)

A professional Tkinter-based desktop E-Commerce application with a modern UI, MVC architecture, and SQL Server integration.

## Features

- **Modern UI**: Hero section, Treeview cart, responsive layouts.
- **User Management**: Login, Signup, Profile, Password Update.
- **Shopping**: Product browsing, Cart management, Checkout flow.
- **Architecture**: MVC/MVP pattern with Dependency Injection.

## Setup

1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Configure `src/data/db.py` with your SQL Server details.
4. Run `python run.py` (or `run_app.bat`).

## Testing

Run automated tests:

```bash
pytest
```
